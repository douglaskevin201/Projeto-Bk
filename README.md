# Projeto BK' - Experiência Imersiva (Álbum ICARUS) 

Este projeto é um site temático desenvolvido para explorar as fronteiras do **HTML5** e **CSS3**. O objetivo foi criar uma interface que refletisse a estética visual dos álbuns do rapper BK', focando em design imersivo e performance.

O menu principal ainda não esta finalizado.

## Tecnologias Utilizadas

* **HTML5 Semântico:** Estruturação focada em acessibilidade e SEO.
* **CSS3:** Uso de gradientes, filtros de desfoque (Glassmorphism), e posicionamento fixo.
* **Design Responsivo:** Layout adaptável para diferentes tamanhos de tela (Mobile First).
* **Integração de Mídia:** Incorporação de vídeos do YouTube via Iframe com controle de aspect-ratio.

## Desafios Técnicos Superados

* **Efeito Parallax:** Implementação de fundos fixos (`background-attachment: fixed`) para criar profundidade durante a rolagem.
* **UI Dinâmica:** Criação de botões flutuantes de navegação com transições suaves para melhorar a experiência do usuário (UX).
* **Estilização de Tipografia:** Uso de fontes externas via Google Fonts para manter a identidade visual do álbum.
